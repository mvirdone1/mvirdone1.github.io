Tanzu Labs Coaching Workshop Tool

This tool contains a collection of coaching prompts for the workshop. How to use:
  • Each teammate should open the site in their own local browser window (concurrently)
  • The coach changes the visibility to "Coach" and then selects a prompt (or uses a random prompt
  • The coach shares the prompt id with the other teammates and each teammate pulls up that prompt ID so everyone can work on the same scenario (set visibility to "Both")
